# Sports Center IFCN

Repositório utilizado para armazenar o código do Sports Center IFCN. Este projeto está sendo desenvolvido como método de avaliação da disciplina de Projeto Integrador, presente no 3º período do curso técnico integrado em Informática, do IFRN *Campus* Currais Novos.

## Ferramentas

As ferramentas utilizadas na aplicação são mostradas na seguinte tabela, juntamente com as suas respectivas versões vigentes.

| Tecnologia | Versão |
|------------|--------|
| Python     | 3.12.0 |
| Django     | 4.2.7  |
| HTML       | 5      |
| CSS        | 3      |

## Padronização

As páginas HTML e as estilizações CSS devem ser armazenadas **somente** na pasta `pages`, e as imagens em `pages/img`. A criação de múltiplos arquivos ou diretórios é de livre escolha de cada desenvolvedor.
